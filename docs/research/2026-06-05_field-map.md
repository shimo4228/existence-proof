## 1. 理論: 「個人」に validate された framework は実在する【確度: 高】

- **Personal Brand Equity (PBE) の3次元** — appeal / differentiation / recognition。Keller (1993) の product brand-equity 次元を個人に転用 (relabel) したもので、**2つの独立した実証研究**が同一構造を確認:
  - Gorbatov et al. 2021, *Personnel Psychology* (N=3,273・7サンプル・EFA/CFA) — https://onlinelibrary.wiley.com/doi/full/10.1111/peps.12412
  - MDPI 2025 (10 interviews + 396 surveys, mixed-methods, Cronbach α >0.7) — https://www.mdpi.com/2076-3387/15/4/148
  - 「製品前提の理論か個人適用可か」への答え: **転用済み・個人で再検証済み**
- **5次元 positioning framework** — strategy / capability / differentiation / audience / performance の5軸・32指標。SEM で validate (RMSEA 0.035)。
  - Shafiee et al. 2020, *European Research on Management and Business Economics* — https://www.sciencedirect.com/science/article/pii/S2444883419300567
  - caveat: 単一研究・イラン文脈。「field consensus」ではなく「validate された一モデル」
- **個人の差別化は製品と働き方が違う** — 新属性の追加ではなく「audience が価値を置く質を、より高いレベルで持つ」ことで立つ。celebrity/executive 限定ではなく誰にでも適用可能 (agentic capability)。
  - Gorbatov, Khapova & Lysova 2018, *Frontiers in Psychology* (100+ 論文 systematic review) — https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2018.02238/full

## 2. authenticity との緊張は実在する — ただし「扱える設計課題」【確度: 高】

- Gershon 2016, *HAU: Journal of Ethnographic Theory* 6(3):223-246: personal branding は「coherent で authentic な self の**印象**を作る semiotic 実践」— 戦略的構築と authentic 表出のギャップを本質的に抱える。Pagis & Ailon 2017 (*Work and Occupations*) が独立に corroborate。 — https://www.journals.uchicago.edu/doi/full/10.14318/hau6.3.017
- 反論側 (HBS blog 等) も「緊張を reconcile する」立場であり、**緊張の存在自体は両陣営が認める**。
- 含意: authenticity-first を捨てる必要はないが、緊張を無視した branding 設計は理論的に破綻する。framework を作るならこの緊張を明示的な設計課題として組み込む。

## 3. AI 時代の核心 finding: LLM はブランド情報の大半を「owned の外」から取る【確度: 高・最重要】

- Omniscient Digital (2026-01-08): 23,387 unique citations / 240 branded prompts / 5 engines (ChatGPT, Perplexity, Gemini, AI Mode, AI Overviews) の分析で、LLM citation の **48% が earned media** (editorial 16% / forums・social 11% / review sites 11% / directories 10%)、commercial 30%、**owned 23%**。 — https://beomniscient.com/blog/how-llms-source-brand-information/
- 3つの独立データセットが方向を corroborate: Muck Rack (84% earned)、別分析 (94%)、Stacker/Scrunch 2025-12 (3rd-party distribution で 4.4x citation lift)。48% は cluster 内で最も保守的な数値。
- caveat: vendor blog 由来で commercial bias はあるが、非提携データと convergent。
- **含意 (本調査で最も load-bearing)**: authorship-strategy (repo / Zenodo / HF = すべて owned channel) には測定された盲点がある。LLM-mediated discovery 自体が earned channel に大きく依存する。**人間向け branding は authorship-strategy と競合せず、それを給餌する補完関係** — 人間が earned channel に書く → LLM がそこを引用する。

## 4. 実践コンセンサス: authority は IDE の外で作られる【確度: 中】

- 2024-2026 の実務側の収束: technical authority は coding 行為そのものではなく、意図的・反復的な public demonstration (技術ブログ / OSS / 登壇 / 持続的 presence) で構築される。
- 差別化要因は volume / virality ではなく **authenticity・clarity・demonstrated competence** (LinkedIn 2026 algorithm 分析等とも convergent)。
- 出典: https://medium.com/@mehdibafdil/building-technical-authority-how-developers-establish-credibility-2abf851d2311 / https://www.productivepassions.com/post/founder-visibility-2026
- caveat: blog 品質ソース。RCT 級の定量証拠なし。なお BAFDIL 記事内の「16%/84%」統計は検証で棄却済み (この finding はそれに依存しない)。

## 5. ツール空白: 「個人/研究プログラム向け authenticity-first branding skill」は存在しない【確度: 高】

- [onvoyage-ai/gtm-engineer-skills](https://github.com/onvoyage-ai/gtm-engineer-skills) (1.1k★, MIT, ~2026-02): GEO/AEO website 最適化の ~12-skill suite。ただし **org/business GTM 前提** (company positioning, team coordination, backlinks)
- coreyhaines31/marketingskills (60+ skills, product/B2B), anthropics/skills の brand-guidelines (視覚 identity 適用のみ), alirezarezvani/claude-skills (C-suite/growth persona 前提) — いずれも個人・研究プログラム向けではない
- 主要 repo 4つの確認に基づく。網羅調査ではないため「空白」は high-confidence だが absolute ではない

## Caveats

- **日本語圏データはゼロ**。全ソース英語圏。Zenn / Qiita / 技術書典での信頼蓄積型 branding の実証は未収集
- earned 48% は 2026-06 時点のスナップショット。AI engine 側の変化で四半期単位で動きうる
- PBE モデルは個人キャリア/employability 文脈で validate されたもの。「研究プログラム」という第3の対象への適用は未検証 (= 弱点であり research gap でもある)
- 軸2 (実践) は blog 品質中心で medium confidence 止まり

## Open Questions

1. 日本語圏 (Zenn / Qiita / note / 技術書典 / 登壇) の信頼蓄積型 branding は英語圏とどう構造が異なるか。日本語圏ではどのプラットフォームが LLM citation 源になるか
2. 「人間向け branding」と「LLM 向け diffusion」を同一コンテンツで両立させると cloaking / 品質劣化が起きるか (memory `human-channel-seo-direction-2026-05` の Part B 適用判断に直結)
3. authenticity-first を維持したまま branding する operational pattern — 理論は緊張の存在を示すが、検証済みプラクティスは学術に乏しい。非マネタイズ branding の個人研究者レベルの先行事例
4. PBE 3次元 / 5次元 positioning は「研究プログラム」(個人キャリアでも商業でもない第3の対象) に適用できるか、固有の次元が必要か

## 次のアクション候補

1. **戦略 doc 化** (推奨・軽量) — 対象確定後、earned-channel 補完戦略を ADR/doc に落とす。skill 化の前に判断の正本が要る
2. **skill 化** — authorship-strategy の sibling として `branding-strategy` skill。ecosystem 空白は確認済み。対象確定が先
3. **既存 skill 導入** — gtm-engineer-skills は org 前提で fit が低い。導入は非推奨、設計の参考に留める
4. **(番外) 第6の研究 line 化** — 「PBE / positioning モデルは idea-diffusion 文脈に適用できるか」は未踏の research gap。authorship-strategy が line になったのと同じ道
